# StudyNook 📚

StudyNook is a modern full-stack study room booking platform where users can discover, add, and reserve study rooms easily. Built with Next.js, Express.js, MongoDB, and Better Auth.

---

## 🚀 Live Links

### Client

https://studynook-client-one.vercel.app

### Server

---

# ✨ Features

- 🔐 Google Authentication with Better Auth
- 🏠 Add New Study Rooms
- ✏️ Update Room Information
- ❌ Delete Rooms
- 📅 Book Study Rooms
- ⏰ Time Slot Booking System
- 🚫 Prevent Double Booking
- 💰 Automatic Booking Cost Calculation
- 📋 Manage Personal Bookings
- 🎨 Modern Responsive UI
- 🌌 Animated Glassmorphism Design
- ⚡ Fast Performance with Next.js
